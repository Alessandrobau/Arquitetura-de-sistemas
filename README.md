## Subir aplicação

- Instalar prisma

```bash
npm install @prisma/client
```

- Migrate

``` bash
npx prisma migrate dev
```

- Subir banco docker

 ``` bash
sudo docker compose up -d 
```

- start application

``` bash
node app.js start
```
